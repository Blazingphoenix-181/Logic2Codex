# **App Name**: CodeStorm

## Core Features:

- Question Input: Allow users to input coding questions and descriptions.
- Logic Input: Provide a section for users to input their logic for solving the problem, triggered by a button.
- Code Generation: Generate code in the desired language (C, C++, Python, Java) based on the provided logic or question, powered by AI tool.
- Language Selection: Implement a button to select the target programming language (C, C++, Python, Java).
- Light/Dark Theme: Allow users to switch between light and dark themes for optimal viewing experience.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey intelligence and innovation.
- Background color: Light gray (#EEEEEE) for light mode and dark gray (#333333) for dark mode.
- Accent color: Teal (#009688) to highlight important elements and interactive components.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, for both headlines and body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist, vector-based icons for language selection and theme switching.
- Use subtle transitions and animations to provide a super smooth user experience.
